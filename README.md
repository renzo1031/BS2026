## ⚡ 快速开始

以 `001` 项目为例：

```powershell
# 1. 初始化数据库
Get-Content -LiteralPath ".\sql\001\init.sql" | mysql -uroot -proot

# 2. 启动后端
Set-Location ".\001-backend"
mvn spring-boot:run

# 3. 启动前端
Set-Location "..\001-frontend"
npm install
npm run dev -- --host 127.0.0.1
```

常用地址：

```text
前端：http://127.0.0.1:5173/
后端：http://localhost:8080
```

## 💬 购买前常见问题

**Q: 项目是否可以直接运行？**  
A: 项目整理时会尽量完成基础运行验证，并提供源码、数据库脚本和说明文档。由于不同电脑的 JDK、MySQL、Node.js 版本可能不同，购买后可协助排查启动问题和本地环境配置。

**Q: 是否支持功能定制？**  
A: 支持。购买项目后可以继续定制页面、业务流程、数据库字段或新增功能模块，也可以按学校要求调整题目方向。

**Q: 是否提供论文 / PPT 支持？**  
A: 可以提供论文思路、系统功能说明、答辩 PPT 内容建议和演示流程梳理，让项目更适合毕业答辩展示。

**Q: 不确定哪个项目适合自己怎么办？**  
A: 可以先说明专业方向、学校要求和想做的题目类型，再推荐更合适的项目或定制方案。
