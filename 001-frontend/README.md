# 校园失物招领中心前端

这是 `001` 校园失物招领中心的 Vue 3 前端项目，包含前台门户、个人中心、保管处理端和管理后台。

## 技术栈

- Vue 3
- Vite
- TypeScript
- Vue Router
- Pinia
- Axios
- Element Plus

## 启动

```powershell
npm install
npm run dev -- --host 127.0.0.1
```

默认访问：

```text
http://127.0.0.1:5173/
```

后端默认地址：

```text
http://localhost:8080
```

## 构建

```powershell
npm run build
```

构建时如果出现第三方包 `#__PURE__` 注释警告或 chunk size 警告，不影响本项目运行。
